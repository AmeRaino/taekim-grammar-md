# 053. ・〜てしかたがない / 〜てしょうがない

- Note ID: 1736184255928
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- ① 我慢できないくらい～だ 気持ちが強く、抑えることができないことを表す。 ② 最高に〜だ

## Structure

V（て形） + しかたがない イAくて + しかたがない ナAで + しかたがない

## Examples

1. 大学受験の結果が気になってしかたがない。
2. ゲーム終了１分前に逆転されてしまい、負けてしまった。本当に悔しくてしかたがない。
3. 第一志望の大学に合格できたので、嬉してしかたがありません。
4. 会社の飲み会が嫌でしかたがない。
5. 海外で一人暮らしを始めたばかりなので、寂しくてしかたがありません。
6. 私の国で大きな地震があったようで、家族と現在連絡がとれません。本当に心配でしかたがありません。
