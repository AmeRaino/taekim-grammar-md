# 025. ・〜ことはない

- Note ID: 1736184255900
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- /

## English

- no need

## Structure

V（辞書形）+ことはない

## Examples

1. たくさん頑張ったんだから、できなくてもがっかりすることはないよ。
   - たくさん頑張ったんだから、できなくてもがっかりすることはないよ。
2. 彼は顔は怖いけど、とても優しいから怖がることはないよ。
   - 彼は顔は怖いけど、とても優しいから怖がることはないよ。
3. コンビニまで歩いて３分だし、人も少ないし、わざわざ化粧していくことはないんじゃない。
   - コンビニまで歩いて３分だし、人も少ないし、わざわざ化粧していくことはないんじゃない。
4. 来週、また帰って来るんだから、わざわざ空港まで見送りに来ることはないよ。
   - 来週、また帰って来るんだから、わざわざ空港まで見送りに来ることはないよ。
