# 010. ・〜かける / 〜かけの

- Note ID: 1736184255885
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- ① 途中まで～して、まだ～し終わらない ② 〜しそうになる。 継続動詞と接続すると①の途中までの状態であることを表す。 瞬間動詞と接続すると②の直前を表す意味。

## English

- ① Started doing something but haven't finished yet ② Almost about to do something

## Structure

V（ます形）ます + かける V（ます形）ます + かけのN

## Examples

1. テーブルの上に飲みかけのコーヒーがおかれている。
2. 息子はやりかけの宿題をおいて、遊びに出かけてしまった。
3. 「壊れかけのradio」は徳永英明の名曲だ。
4. 下痢が治りかけていたのに、何か変な物を食べたのか、またお腹が痛くなってきた。
5. 母は夕食を作りかけて、今ドラマに夢中だ。
6. 書きかけのレポートを今日中に仕上げるつもりだ。
