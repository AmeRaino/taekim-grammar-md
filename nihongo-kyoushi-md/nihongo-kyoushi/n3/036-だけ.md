# 036. ・〜だけ

- Note ID: 1736184255911
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- ～の範囲は全部て 「～の範囲の限界まで～する」と言いたいときに使う。

## English

- as much as

## Structure

普通形（肯定形）＋ だけ ※Nにはつかない。

## Examples

1. できるだけ水を飲むようにしてください。
   - Please try to drink water as much as possible.
2. どうぞ、好きなだけ食べてくださいね。
   - Please eat as much as you want.
3. やれるだけのことはやったので、もう後悔はありません。
   - I've done all I can do, so I have no regrets.
4. 泣きたいときは、我慢せず泣きたいだけ泣けばいいんですよ。
   - If you want to cry, don't hold back, just cry as much as you want.
