# 043. ・〜だらけ

- Note ID: 1736184255918
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- "〜がたくさんある / 〜がたくさんついている" 同質のものが多量に存在していてよくない状態。

## English

- Indicates that something has a lot of something similar, which is undesirable.

## Structure

N + だらけ

## Notes

- Nには必ずしも悪い・汚いイメージの物が来るとは限らない。 例：なんだこの、絵画だらけの部屋は。

## Examples

1. 私が翻訳した英語は間違いだらけだったので、友達に直してもらった。
2. 自信があったのに、先週のテストは間違えだらけだった。
3. この車はもう傷だらけですが、たくさんの思い出が詰まっているので、まだ買いかえたくないです。
4. 息子が傷だらけで帰ってきた。友達とケンカしたようだ。
5. 彼の部屋は漫画だらけで、勉強の本は全然なかった。
6. 彼の言うことは矛盾だらけで、全く信用できない。
