# 023. ・〜ことから

## Meaning

- 〜が理由で / 〜が由来で / 〜が判断の根拠になって

## English

- from the fact that

## Structure

V（普通形）+ ことから イA（普通形）+ ことから ナAな / である / だった +　ことから Nである / だった + ことから

## Examples

1. 彼は何でも知っていることから、「歩く辞典」と呼ばれている。
   - 彼は何でも知っていることから、「歩く辞典」と呼ばれている。
2. 彼は中国語が話せることから、グローバル部へ配属となった。
   - 彼は中国語が話せることから、グローバル部へ配属となった。
3. 彼の絵は独創的なことから、現代ピカソと呼ばれるようになった。
   - 彼の絵は独創的なことから、現代ピカソと呼ばれるようになった。
4. 彼の絵は独創的なことから、現代ピカソと呼ばれるようになった。
   - 彼の絵は独創的なことから、現代ピカソと呼ばれるようになった。
5. 道路が濡れていることから、雨が降ったということがわかる。
   - 道路が濡れていることから、雨が降ったということがわかる。
6. 彼女は顔色が良いことから、病気ではないと自分で判断したが、それが間違いであった。
   - 彼女は顔色が良いことから、病気ではないと自分で判断したが、それが間違いであった。
