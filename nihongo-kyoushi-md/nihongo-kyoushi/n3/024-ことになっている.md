# 024. ・〜ことになっている

- Note ID: 1736184255899
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- ～という決まりである / 〜することが決まっている 規則や習慣、予定などでそうすること、そうしないことがすでに決まっているという意味。

## English

- Rules, customs or schedules that has been decided

## Structure

V（辞書形 / ナイ形）+ ことになっている

## Examples

1. この会社では、毎朝、仕事の前にラジオ体操をすることになっている。
2. この国では、車は右車線を走ることになっている。
3. この店には女性しか入れないことになっている。
4. 来週の金曜日は東京へ出張へ行くことになっている。
5. 燃えるゴミは月・水・金に出すことになっている。
6. この会社では、毎週水曜日は残業してはいけないことになっている。
