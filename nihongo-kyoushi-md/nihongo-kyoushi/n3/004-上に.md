# 004. ・〜上に

- Note ID: 1736184255879
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- ～に加えて / それに

## English

- on the top of that / in addition

## Structure

普通形 + 上に ・V（普）+上に ・イA（普） + 上に ・ナAな + 上に ・ナAである + 上に ・Nの + 上に ・Nである + 上に

## Notes

- 「上に」の前が良いことの場合、「上に」の後ろも良いことが来る。逆に「上に」の前が悪いことの場合、「上に」の後ろも悪いことが来る。

## Examples

1. 最近の携帯は使い方が機能がたくさんある上に、カメラの性能も良いからとても便利だ。
2. このアニメは迫力がある上に泣けるシーンがたくさんあっておすすめだよ。
3. 豊洲の海鮮は安い上に新鮮だ。
4. この会社は待遇が悪い上に仕事の量も多いから３年以内に転職する人が多い。
5. ユニクロは安い上に質もよく多くの人から愛されているブランドだ。
6. この国ではトイレが汚い上にトイレットペーパーがないから最悪だ。
