# 017. ・〜きる

- Note ID: 1736184255892
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- ①全部〜する / 最後まで〜する ②強く〜する / 非常に〜する

## Structure

V（ます形）ます + きる

## Examples

1. 田中さんは足を怪我しながらも、なんとか42.195km走りきった。
2. 勉強熱心なトムさんは、みんなの日本語初級１の単語を３週間で覚えきった。
3. 数えきれないほどの星が空一面に広がっている。
4. 山下さんは疲れきって、熟睡している。
5. 黒田さんは「明日までに絶対に完成させます」と言いきったのに、今朝確認すると全体の50%しか終わっていなかった。
6. 息子は大学を卒業しても働かず、毎日家でYouTubeばかりみていて、私は困りきっている。
