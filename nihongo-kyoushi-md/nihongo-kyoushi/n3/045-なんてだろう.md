# 045. ・（なんて）〜だろう

- Note ID: 1736184255920
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- 非常に〜だ 感激したことや驚いたことを気持ちを込めて言う表現。

## Structure

普通形 + だろう

## Examples

1. あぁ、なんて美しい人なんだろう。あんなに綺麗な人に今まであったことがない。
   - Oh, what a beautiful person. I've never met anyone so beautiful in my life.
2. なんて美味しい料理なんだろう。こんな料理なら毎日でも食べられるよ。
   - What delicious food. I could eat this kind of food every day.
3. 初めてインターネットを使った時、なんて便利なんだろうと思いました。
   - When I used the Internet for the first time, I thought how convenient it was.
4. iPhoneを初めて触った時、なんて素晴らしい発明だろうと思いました。
   - When I first touched the iPhone, I thought what a wonderful invention it was.
5. わぁ、なんて軽いのだろう。このノートパソコンなら持ち運びも楽チンだね。
   - Wow, how light it is. This laptop is so easy to carry around.
6. なんて恐ろしい話なんだろう。
   - What a horrible story.
