# 030. ・〜さえ〜ば

- Note ID: 1736184255905
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- 〜が実現できれば、他は問題ない。 ある１つの条件が満たされれば、他は問題にならない、大丈夫だと言う意味。

## English

- Presupposes an only or particular condition. This sentence pattern indicates that as long as one specific condition is met, everything else is not an issue and will be fine.

## Structure

V（ます形）ます + さえあれば イAく + さえあれば ナAで + さえあれば N + さえあれば

## Examples

1. 田中さんは暇さえあれば、本を読んでいます。
2. 薬を飲みさえすれば、病気が治るというものではありません。
3. あの時、話し合ってさえいれば、妻と別れることはなかったのに。
4. お金さえあれば、他には何もいらない。
5. この子が健康でさえいてくれれば、私は幸せです。
6. 自分さえよければ、他人がどうなってもいいの？
