# 020. ・〜こそ

- Note ID: 1736184255895
- Deck: Nihongo Kyoushi / N3
- Note Type: kyoushi
- JLPT: N3
- Tags: GN3, N3

---

## Meaning

- 強調を表す。 プラスの意味で使われる。（マイナスの意味を強調する使い方はしない。）

## English

- Expresses emphasis. It is that....

## Structure

N + こそ V（辞）+ ことこそ 普通形 + からこそ

## Examples

1. 今年こそJLPTに受かりたい。
2. 来年の冬こそ、スキーに行こう！
3. この本こそ私が探していた本です。ついに見つけました。
4. 健康こそが、私の一番の財産だ。
5. 彼こそこの会社にふさわしい人だ。
6. 毎日続けてジョギングをすることこそ、ダイエットの秘訣だ。
