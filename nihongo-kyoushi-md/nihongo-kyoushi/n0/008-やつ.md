# 008. ・〜やつ

- Note ID: 1736184346806
- Deck: Nihongo Kyoushi / N0
- Note Type: kyoushi
- JLPT: N0
- Tags: GN0, N0

---

## Meaning

- 〜というもの 物を詳しく説明するときに使う表現。 カジュアルな言い方。

## Structure

文 + やつ

## Examples

1. A：日本語の辞書を探してるんだけど、何かおすすめある？ B：これなんかどう？ A：うーん、アクセントが書かれているやつが欲しいんだけど。
   - A: I'm looking for a Japanese dictionary, do you have any suggestions? B: How about this one? A: Hmmm, I need one with accents on it.
2. A：このコートいいんじゃない。 B：そうだね。でも、もう少しゆったりしたやつが欲しいなあ。
   - A: I think this coat is good. B: Yes, it is. But I'd like something a little more loose.
3. A：あの曲って何っていう名前だっけ？ B：あれって、どれ？ A：あれだよ、あれ。えーっと、最近若い人に人気があって、よくCMで流れているやつ。
   - A: What's the name of that song again? B: Which one is that? A: That one, that one. It's popular among young people these days, and is often shown in commercials.
4. もっと大きめのやつないの？
   - Isn't there a bigger one?
