# 065. ・〜ようと思う

- Note ID: 1736184039576
- Deck: Nihongo Kyoushi / N4
- Note Type: kyoushi
- JLPT: N4
- Tags: GN4, N4

---

## Meaning

- 話し手が発話の時点で考えた計画や決心を述べる時に使う表現。

## English

- This sentence pattern is used to talk about the speaker’s determinations or plans.

## Structure

V（意向形） + と思う

## Examples

1. これから、図書館で勉強しようと思います。
   - I'm going to study at the library.
2. 今日は給料日なので、欲しかったゲームを買おうと思います。
   - Today is payday, so I'm going to buy the game I want.
3. 早く仕事が終わったので、ちょっと飲みに行こうと思います。
   - I finished work early, so I'm going out for a drink.
4. 風邪を引いたので、今日はゆっくり家で休もうと思います。
   - I caught a cold, so I'm going to take it easy at home today.
5. 明日は早く起きようと思います。
   - I'm going to wake up early tomorrow.
6. 忘れ物をしたので、一度家に帰ろうと思います。
   - I forgot something, so I'm going to go home once.
7. 夏休みはアルバイトを増やそうと思います。
   - I'm going to get another part time job during the summer vacation.
8. お年玉でゲームを買おうと思います。
   - I'm going to buy a video game with my Otoshidama (New Year's allowance).
9. 授業が簡単すぎるので、もう少し難しい本を買って自分で勉強しようと思います。
   - I'm going to buy some more difficult books and study by myself because the classes are too easy.
10. 来年はJLPTを受けようと思います。
   - I'm going to take the JLPT next year.
