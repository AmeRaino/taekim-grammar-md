# 038. ・〜と

- Note ID: 1736184039549
- Deck: Nihongo Kyoushi / N4
- Note Type: kyoushi
- JLPT: N4
- Tags: GN4, N4

---

## Meaning

- ある動作を行うと、必ず、別のことがらや作用が続いて起こることを表す。 機械の使い方や道順の説明によく使われる。

## English

- "whenever A happens, B also happens."

## Structure

V（辞書形 / ナイ形）+ と イAい + と ナAだ + と Nだ + と

## Notes

- ・後件に意志を表す文や依頼表現などは使えない。 ・仮定条件には使うことができない。 例：もし１００万円あると、何がしたいですか？

## Examples

1. この道をまっすぐ行くと、病院があります。
2. このまままっすぐ行って、２つ目の角を左に曲がると、交番があります。
3. お金を入れて、食べたい料理のボタンを押すと、食券が出ます。
4. 春になると桜が咲きます。
5. 冬になると、暗くなるのが早くなります。
6. 一日中パソコンを使うと、目が痛くなります。
7. 私は彼の顔を見ると、いつも元気になります。
8. 私は電気を消さないと、寝られません。
9. このダイアルを回すと、音が大きくなります。
10. このボタンを押すと、ドアが開きます。
