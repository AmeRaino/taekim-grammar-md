# 048. ・〜ば

- Note ID: 1736184039559
- Deck: Nihongo Kyoushi / N4
- Note Type: kyoushi
- JLPT: N4
- Tags: GN4, N4

---

## Meaning

- XばY：Xという条件が成立すると、Yが起こることを表す。

## English

- "if X then Y" If condition X is met, then Y will happen.

## Structure

V（ba form） イA + ければ / イA + なければ ナA + ならば / ナA + でなければ N + ならば / N ＋でなければ

## Notes

- ① 実現性の高い条件、低い条件、どちらの場合でも使える。 ② 後件には良い結果が来ることが多い。 ③ 少し硬い表現で書き言葉的。 ④ ことわざで使われることが多い。（例：三人よれば、文殊の知恵。）

## Examples

1. 春になれば、桜が咲きます。
   - When spring comes, the cherry blossoms will bloom.
2. １００万円当たれば、車を買います。
3. 次の電車に乗れば、間に合います。
4. トムさんに聞けば、なんでも教えてくれますよ。
5. 毎日練習すれば、日本語が上手になりますよ。
6. この道を行けば、問題ないだろう。
7. 機会があれば、行ってみたいです。
8. 仕事が早く終われば、飲み会に参加できます。
9. もし明日、雨が降れば、試合は中止です。
10. あと３,０００円安ければ、買います。
