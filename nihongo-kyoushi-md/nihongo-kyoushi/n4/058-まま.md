# 058. ・〜まま

- Note ID: 1736184039569
- Deck: Nihongo Kyoushi / N4
- Note Type: kyoushi
- JLPT: N4
- Tags: GN4, N4

---

## Meaning

- 同じ状態が変わらずに続くことを表す。

## English

- "while" Used to show the same situation continues.

## Structure

V（タ形 / ナイ形） + まま イAい + まま ナAな + まま Nの + まま

## Examples

1. 昨日は電気をつけたまま寝てしまった。
2. どうぞ、靴を履いたままでお上がりください。
3. 友達に本を借りたまま、まだ返していません。
4. 見て、ジョンさん、目を開けたまま寝てる。
5. 息子は朝、家を出たまま、まだ帰ってこない。
6. この靴は買ってから一度も履いていないので、新しいままだ。
7. 電子レンジで温めてから、だいぶ時間が経つのに、まだ温かいままだ。
8. 私の妻は１０年前も今も綺麗なままだ。
9. この街は昔と変わらず不便なままだ。
10. これは生のままでも食べられますか。
