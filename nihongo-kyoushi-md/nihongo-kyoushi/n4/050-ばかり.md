# 050. ・〜ばかり

- Note ID: 1736184039561
- Deck: Nihongo Kyoushi / N4
- Note Type: kyoushi
- JLPT: N4
- Tags: GN4, N4

---

## Meaning

- 同じ物がたくさんある、同じことを何度もするという意味を表す。 マイナスの意味で使われることが多い。

## English

- It refers to having a lot of the same thing, or doing the same thing repeatedly. It's often used in a negative sense.

## Structure

V（て形） + ばかり N + ばかり

## Examples

1. 友達はいつもゲームしてばかりいる。
2. 田中さんはいつも仕事中に寝てばかりいます。
3. 私の父は休みの日にいつも、テレビを見てばかりいます。
4. 食べてばかりいると、太りますよ。
5. 遊んでばかりいないで、勉強もしなさい。
6. 毎日お酒を飲んでばかりいると、いつか体を壊しますよ。
7. 座ってばかりなので、お尻が痛いです。
8. 山田先生はいつも怒ってばかりで、授業が楽しくない。
9. この会社の社員は若い女性ばかりだ 。
10. 友達は旅行中、セルフィーばかりしていました。
