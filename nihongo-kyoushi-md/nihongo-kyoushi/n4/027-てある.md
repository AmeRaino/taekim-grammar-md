# 027. ・〜てある

- Note ID: 1736184039538
- Deck: Nihongo Kyoushi / N4
- Note Type: kyoushi
- JLPT: N4
- Tags: GN4, N4

---

## Meaning

- 誰かが何か目的があってその行為を行い、その行為の結果が残っていることを表す文型。

## English

- somebody has done something intentionally and that state continues.

## Structure

V（て形） + ある

## Notes

- 動詞は他動詞が来る。

## Examples

1. 壁にポスターが貼ってあります。
2. テーブルにお皿が並べてあります。
3. もう宿題はしてあります。
4. お風呂はもう沸かしてあります。
5. あ、机の上にケーキが置いてある。
6. もうホテルの予約はしてありますか？
7. 昼ごはんは用意してあるから、電子レンジでチンして食べてね。
8. 昼ごはんは用意してあるから、電子レンジでチンして食べてね。
9. ん？机の上に何かメモが置いてあるなあ。
10. この問題の答えは１２０ページに書いてありますよ。
