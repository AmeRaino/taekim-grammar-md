# 036. ・〜てみる

- Note ID: 1736184039547
- Deck: Nihongo Kyoushi / N4
- Note Type: kyoushi
- JLPT: N4
- Tags: GN4, N4

---

## Meaning

- いいかどうかわからないので実際に行動して確かめる

## English

- try doing something

## Structure

V（て形）+ みる意志動詞と接続する。

## Examples

1. 似合うかどうかわからないので、履いてみます。
   - I don't know if it suits me, so I'll try it on.
2. サイズが合うかどうかわからないので、着てみます。
   - I don't know if it fits, so I'll try it on.
3. 美味しいかどうかわからないので、試しに食べてみます。
   - I don't know if it tastes good, so I'll try it.
4. もしお金がたくさんあったら、何をしてみたいですか。
   - If you had a lot of money, what would you like to do?
5. 日本に行ったら、相撲を見てみたいです。
   - If I went to Japan, I would like to try watching sumo wrestling.
6. 一度でいいから海が綺麗な南の島でのんびり過ごしてみたいなぁ。
   - I would like to spend a relaxing time on a southern island with a beautiful sea just once.
7. 京都に行ったら、着物を着てみたいです。
   - When I go to Kyoto, I want to try on a kimono.
8. 大人になったら、世界一周旅行をしてみたいです。
   - When I grow up, I want to try traveling around the world.
