# 015. ・〜こととて

- Note ID: 1736184315700
- Deck: Nihongo Kyoushi / N1
- Note Type: kyoushi
- JLPT: N1
- Tags: GN1, N1

## Meaning

- "〜だから / 〜なので"

## English

- Used to express a reason for an apology.

## Structure

V（普通形）＋こととて イA（普通形）＋こととて ナA（普通形）な＋こととて 名詞の＋こととて

## Notes

- ① 古い表現。 ② 動詞の否定形と接続する場合「〜ないこととて」と「〜ぬこととて」の２通りの言い方がある。

## Examples

1. 慣れないこととて、あまり上手に作れませんでした。
   - 慣れないこととて、あまり上手に作れませんでした。
2. 子供のやったこととて、どうや許してやってください。
   - 子供のやったこととて、どうや許してやってください。
3. 突然のこととて、まだ準備ができておりません。
   - 突然のこととて、まだ準備ができておりません。
4. プレゼンは初めてのこととて、緊張してしまいうまく話せなかった。
   - プレゼンは初めてのこととて、緊張してしまいうまく話せなかった。
5. そんなの昔のこととて、覚えておりませんよ。
   - そんなの昔のこととて、覚えておりませんよ。
