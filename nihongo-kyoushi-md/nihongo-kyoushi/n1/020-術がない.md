# 020. ・〜術がない

- Note ID: 1736184315705
- Deck: Nihongo Kyoushi / N1
- Note Type: kyoushi
- JLPT: N1
- Tags: GN1, N1

---

## Meaning

- 〜する方法がない / 〜することができない

## English

- there is no way / cannot do

## Structure

V（辞書形）＋ 術がない

## Notes

- 「術（すべ）」は「方法」のこと

## Examples

1. 試せることは全て試したし、もうこれ以上なす術がない。
2. KFCのレシピは極秘なので、たとえ社員になっても知る術がないらしい。
3. 電話番号やメールアドレスもわからないし、SNSも使っていないし、連絡する術がない。
4. 敵は空から攻撃してくるので、我々はもう戦う術がない。
