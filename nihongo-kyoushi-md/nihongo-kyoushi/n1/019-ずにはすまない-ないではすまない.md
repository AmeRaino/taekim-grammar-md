# 019. ・〜ずにはすまない / 〜ないではすまない

- Note ID: 1736184315704
- Deck: Nihongo Kyoushi / N1
- Note Type: kyoushi
- JLPT: N1
- Tags: GN1, N1

---

## Meaning

- "〜ないでは許されない / 必ず〜しなければならない"

## English

- "must [do something] / It is necessary to [do something]" Used to convey that, given the circumstances or social norms, one is obligated or required to perform a certain action. It implies that failing to do so would be unacceptable.

## Structure

V（ナイ形）ない + ずにはすまない　※「する」は「せずにはすまない」となる。 V（ナイ形）+ ではすまない

## Notes

- ①後件に意志性のある表現や、命令、否定を表す表現は使えない。 ②硬い表現で書き言葉として使うことが多い。 ③過去のことについてしか使えない。

## Examples

1. 借りたカメラを壊してしまったので、弁償せずにはすまない。
2. 薬だけでは直すことができないので、手術せずにはすまない。
3. せっかく私のために開いてくれたパーティーなので参加せずにはすまなかった。
4. 不祥事を起こしてしまったので、責任を取らずにはすまないだろう。
5. 母の大事なコーヒーカップを割ってしまったので、叱られずにはすまないだろう。
6. 冗談のつもりだったけど、相手を傷つけてしまったのなら謝らないではすまないだろう。
