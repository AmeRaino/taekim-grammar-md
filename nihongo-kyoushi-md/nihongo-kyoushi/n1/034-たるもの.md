# 034. ・〜たるもの

- Note ID: 1736184315719
- Deck: Nihongo Kyoushi / N1
- Note Type: kyoushi
- JLPT: N1
- Tags: GN1, N1

---

## Meaning

- 〜の立場にあるのだから、（当然）・・・だ。

## English

- As someone/something that is [role/status], it is expected to [quality/behavior].

## Structure

N + たる

## Notes

- 後件には「〜なければならない」や「〜べきだ」のような判断の表現が来る。

## Examples

1. 日本人たるもの、おもてなしの心を忘れてはならない。
   - 日本人たるもの、おもてなしの心を忘れてはならない。
2. 剣士たるもの、敵に背中を見せてはならない。
3. 医者たるもの、患者の命を優先に考えるべきだ。
4. 政治家たるもの、発言には慎重にならなければならない。
5. 教師たるもの、学生の見本となるような行動すべきだ。
