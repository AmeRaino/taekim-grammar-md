# 032. ・〜たら〜たで

- Note ID: 1736184315717
- Deck: Nihongo Kyoushi / N1
- Note Type: kyoushi
- JLPT: N1
- Tags: GN1, N1

---

## Meaning

- 〜の場合は、当然・・・すべきだ / 〜の場合は、必ず・・・という結果になる

## English

- "In the case of 〜, it is natural that ... should be done / In the case of 〜, it will inevitably lead to ... as a result."

## Structure

V（タ形）+ ら ＋ V（タ形） ＋ で イAけかったら ＋ イAかった ＋ で ナAなら ＋ ナAな　＋ で Nなら ＋ N ＋ で

## Examples

1. 試験に落ちたら落ちたで、また来年受けるだけだ。
   - 試験に落ちたら落ちたで、また来年受けるだけだ。
2. 失敗したらしたで、もう一度チャレンジすればいい。
   - 失敗したらしたで、もう一度チャレンジすればいい。
3. 学校に行くのは面倒だが、行ったら行ったで楽しい。
   - 学校に行くのは面倒だが、行ったら行ったで楽しい。
4. 時々、息子に会いたいが、会ったら会ったでうるさい。
   - 時々、息子に会いたいが、会ったら会ったでうるさい。
5. もちろん、安く買えるほうがいいが、安かったら安かったで少し不安になる。
6. 恋人が欲しいが、いたらいたで自由な時間もなくなるしなあ・・・。
7. 仕事、暇そうでいいねなんて言われますが、暇なら暇で眠くなるし、つまらないし、ストレスもたまりますよ。
