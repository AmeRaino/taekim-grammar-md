# 076. ・〜にはあたらない

- Note ID: 1736184315761
- Deck: Nihongo Kyoushi / N1
- Note Type: kyoushi
- JLPT: N1
- Tags: GN1, N1

---

## Meaning

- ① 〜するほど価値や重みがない / 〜する必要がない② 〜には該当しない

## English

- ① It is not worth ~ / There is no need to ~.② It does not fall under ~ / It does not qualify as ~.

## Structure

V（辞書形）+ にはあたらないN + にはあたらない※Nはする動詞のN

## Notes

- ①「あたる」とは「相当する」といみ意味②「〜には」の「は」は省略可

## Examples

1. 私の国では１時間の遅刻は驚くには当たりません。
   - 私の国では１時間の遅刻は驚くには当たりません。In my country, being an hour late is not something to be surprised about.
2. 彼の実力からいって、JLPT N1に合格しても驚くには当たりません。
   - 彼の実力からいって、JLPT N1に合格しても驚くには当たりません。
3. 人気監督の新作映画は思った以上にレビューが低く、見るに当たらないそうです。
   - 人気監督の新作映画は思った以上にレビューが低く、見るに当たらないそうです。
4. ちょっと怒られたぐらいで、そこまで落ち込むに当たらないよ。
   - ちょっと怒られたぐらいで、そこまで落ち込むに当たらないよ。
5. 彼のやったことは犯罪には当たらないそうだ。
   - 彼のやったことは犯罪には当たらないそうだ。
