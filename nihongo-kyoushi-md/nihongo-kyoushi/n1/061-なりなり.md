# 061. ・〜なり〜なり

- Note ID: 1736184315746
- Deck: Nihongo Kyoushi / N1
- Note Type: kyoushi
- JLPT: N1
- Tags: GN1, N1

## Meaning

- 〜でもいいし、〜でもいいし、とにかく何か・・・ 〜でもいいし、〜でもいいし、好きなように・・・

## English

- It can be ~ or ~, just do something... It can be ~ or ~, do as you like...

## Structure

V1（辞書形）+ なり + V2（辞書形）+ なり N1 + なり + N2 + なり

## Notes

- ①「なり」１つだけでも使うことができる ②目上の人には使えない。

## Examples

1. 冷蔵庫に色々入っているから、料理するなり、冷凍食品をチンするなりして食べてね。
   - There are various things in the fridge, so cook something or heat up some frozen food to eat.
2. 煮るなり焼くなり、好きなように調理してください。
3. わからないことがあったら、先生に聞くなり、辞書で調べるなりしてください。
4. 休みで仕事がないなら、掃除するなり洗濯するなり、家事を手伝ったらどう？
5. もったいないのもわかるけど、売るなり捨てるなりしないと物がたまっていく一方よ。
