# 130. 〜ものがある

- Note ID: 1736183958456
- Deck: Nihongo Kyoushi / N2
- Note Type: kyoushi
- JLPT: N2
- Tags: GN2, N2

---

## Meaning

- なんとなく〜だと感じられる。

## Structure

V（辞書形） + ものがある イAい + ものがある ナAな + ものがある

## Notes

- 「ものがある」の前には感情を表す言葉が来ることが多い

## Examples

1. みんながJLPTの試験に受かるとは限らないので、毎回、学生にテストの結果を聞くのは辛いものがある。
   - Not all students will pass the JLPT, so it's hard to ask students about their test results every time.
2. その男の証言には、どこか不自然なものがあった。
3. その男の証言には、どこか不自然なものがあった。
4. プロの演奏を聞いて、何か心に響くものがあった。
5. ジョンさんの日本語の上達の速さには、驚くべきものがある。
6. 今回の地震による被害はひどいものがある。
7. 増税や年金問題などのニュースを見ると、正直辛いものがある。
