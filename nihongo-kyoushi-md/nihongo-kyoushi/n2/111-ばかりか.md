# 111. 〜ばかりか

- Note ID: 1736183958437
- Deck: Nihongo Kyoushi / N2
- Note Type: kyoushi
- JLPT: N2
- Tags: GN2, N2

---

## Meaning

- 〜だけでなく・・・

## English

- not only 〜 but also・・・"

## Structure

V（普通形）+ ばかりか イA（普通形）+ ばかりか ナAな/である + ばかりか Nである + ばかりか

## Notes

- ①後件には「〜も」、「〜さえ」、「〜まで」などがよく使われる。 ②後件に意志や命令、勧誘などの表現はほとんど来ない。

## Examples

1. トムさんは１０年も日本に住んでいるのに、漢字ばかりかひらがなさえ書けない。
2. ジェームスさんは日本語ばかりか、中国語もペラペラだ。
3. 薬を飲んでいるのに、咳が止まらないばかりか、ひどくなる一方だ。
4. 彼は無断欠席が多いばかりか、出勤しても真面目に働かずスマホばかり触っている。
5. キムさんはスポーツが上手なばかりか、成績も優秀である。
6. 嘘ばかりついていると、信頼を失うばかりか、友達をなくすことになるよ。
7. 初めて料理してみたが、見た目ばかりか、味も最悪だ。
8. 鬼滅の刃という漫画は国内ばかりか、世界中でとても人気がある。
