# 143. ・〜を契機に / 〜を契機として

- Note ID: 1736183958469
- Deck: Nihongo Kyoushi / N2
- Note Type: kyoushi
- JLPT: N2
- Tags: GN2, N2

---

## Meaning

- 〜をいい機会だと考えて / 〜をきっかけにして

## Structure

N + を契機に

## Notes

- ・書き言葉・文の後ろにはプラスの意味の言葉が来ることが多い。

## Examples

1. 転職を契機に、次の会社では自分らしく働こうと決意した。
2. 入院を契機に、今後はできるだけ毎日運動しようと思った。
3. 震災を契機に、改めて防災対策をしっかりしようと家族で話し合った。
4. スペイン人によるフィリピン支配は1521年マゼラン率いるスペイン艦隊がセブ島に到着したことを契機として行われた。
