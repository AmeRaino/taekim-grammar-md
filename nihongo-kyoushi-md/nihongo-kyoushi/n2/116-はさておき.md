# 116. ・〜はさておき

## Meaning

- 〜のことは今は置いておいて

## English

- Setting aside ～ for now Used when changing the topic and getting to the main point.

## Structure

N + はさておき

## Notes

- 話題を変え、本題に入るときなどに使う。

## Examples

1. 冗談はさておき、そろそろ本題に入ろう。
2. 見た目はさておき、味はとても美味しいよ。
3. 誰がミスしたのかはさておき、今後ミスが起きないようにはどうしたら良いかを考えることが大事だ。
4. 誰が発表するかはさておき、先にプレゼンの内容について話し合いましょう。
   - 誰が発表するかはさておき、先にプレゼンの内容について話し合いましょう。
5. 難しい話かはさておき、みんなで楽しくやりましょう。
   - 難しい話かはさておき、みんなで楽しくやりましょう。
6. 何はさておき、君に今の気持ちを伝えたい。
   - 何はさておき、君に今の気持ちを伝えたい。
