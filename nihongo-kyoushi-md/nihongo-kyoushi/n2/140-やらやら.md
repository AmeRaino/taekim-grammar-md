# 140. ・〜やら〜やら

- Note ID: 1736183958466
- Deck: Nihongo Kyoushi / N2
- Note Type: kyoushi
- JLPT: N2
- Tags: GN2, N2

---

## Meaning

- 〜や〜など 色々あるがその中の代表例を１つ２つあげる時に使う。いろいろあって大変だ、不満があるということに使われることが多い。

## English

- and so on

## Structure

V（辞）+ やらイAい + やらN + やら

## Examples

1. 最近、勉強やらバイトやらで毎日忙しい。
   - I've been busy every day lately with studying, part-time work, and other things.
2. 昨日、お酒を飲み過ぎたせいで、頭が痛いやら吐き気がするやらで大変だったよ。
3. １２月は部屋を掃除するやら年賀状を書くやらで忙しい。
4. 大事に育てた娘が結婚することになって嬉しいやら寂しいやら複雑な気持ちだ。
5. さっきコンビニでおにぎりやらお茶やらを買った。
