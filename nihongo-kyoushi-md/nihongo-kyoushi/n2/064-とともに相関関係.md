# 064. ・〜とともに（相関関係）

- Note ID: 1736183958390
- Deck: Nihongo Kyoushi / N2
- Note Type: kyoushi
- JLPT: N2
- Tags: GN2, N2

---

## Meaning

- ～すると、それに応じてだんだん～が変化すると、・・・も一緒に変化する

## English

- When [doing something], gradually in response [something] changes, and [something else] changes along with it.

## Structure

V(辞) + とともに N + とともに

## Examples

1. 時間がたつとともに、手の痛みが引いてきた。
2. インターネットの普及とともに、いつでもどこでも色々な情報が手にはいるようになった。
3. SNSの普及とともに、セルフィーをする人が増えてきた。
4. 春が近づくとともに、少しずつ暖かくなってきた。
