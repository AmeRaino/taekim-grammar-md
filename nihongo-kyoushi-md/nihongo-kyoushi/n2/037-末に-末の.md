# 037. ・〜末に / 〜末の

- Note ID: 1736183958363
- Deck: Nihongo Kyoushi / N2
- Note Type: kyoushi
- JLPT: N2
- Tags: GN2, N2

---

## Meaning

- 色々〜したけど、最終的に。

## Structure

V（タ形） + 末にNの + 末に　※Nはする動詞のN

## Notes

- 末にの「に」は省略可能。

## Examples

1. 色々悩んだ末、もうしばらくこの会社で頑張ってみることにしました。
2. 色々と考えた末、大学を卒業したら国へ帰ることにした。
3. 二年浪人した末に、やっと医学部に合格できた。
4. 試合は巨人と阪神の激しい戦いの末、巨人がサヨナラホームランで勝利した。
5. さんざん検討した末に出した結論ですので、後悔はありません。
6. 妻とはしっかりと話し合った末に、離婚するなった。
7. 試行錯誤の末に、完成した製品がこちらです。
8. 警察は５時間に及ぶ説得の末、何とか銀行に立て篭もる男性を捕まえることができた。
