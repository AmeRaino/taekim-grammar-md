#### 44 When direct object is the topic

Vocabulary
- 1. 日本語 【に・ほん・ご】 – Japanese (language)
- 2. 習う 【なら・う】 (う-verb) – to learn

The direct object particle is different from particles related to location in that you cannot use any other particles at the same time. For example,

going by the previous section, you might have guessed that you can say 「をは」 to express a direct object that is also the topic but this is not the case. A

topic can be a direct object without using the 「を」 particle. In fact, putting the 「を」 particle in will make it wrong.

Examples
- 1. 日本語を習う。
- Learn Japanese.
- 2. 日本語は、習う。
- About Japanese, (will) learn it.

Please take care to not make this mistake.
- 日本語をは、習う。
- (This is incorrect.)
